# Automation Scripts

PowerShell scripts for automating repetitive development tasks.

## Scripts Overview

- **setup-next.ps1:** Script for creating a Next.js website.
- **setup-vite.ps1:** Script for creating a Vite website.
- **initial-commit.ps1:** Script for making an initial commit to a GitHub repository.

## Getting Started

1. Clone this repository: `git clone https://github.com/w11dcard/automation-scripts .`
2. Set environment variables for **GITHUB_TOKEN** & **GITHUB_USERNAME**
3. Set environment variables for each script, for accessing them from anywhere inside you system.
4. Run the scripts, replacing **SCRIPT_NAME** with the script you want to execute: `& $env:SCRIPT_NAME`
5. Follow script instructions.
